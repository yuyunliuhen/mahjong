由于麻将在国外正处于发展期，因此尚未形成非常官方的翻译，因此在一些术语的翻译方面也存在不少诧异。不过本着雅俗共赏的原则，无论翻译是官方性的标配，还是白话英语的口水话，都讲究能让老外看得懂为主。当然，其中不乏一些非常有意思的翻译方式，以下是部分麻将术语的英文翻译：
