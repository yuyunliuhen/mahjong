###询问(胡/杠/碰)
action定义:
GAME_ACTION_PONG:1,
GAME_ACTION_KONG:10,
GAME_ACTION_WIN:100,
GAME_ACTION_CANCEL:1000
{
	"msg_id":1008，
	"action":1111,
	"card_type":0,
	"card_val":1,
	"pong_or_kong_player_index":1,		
	"cur_player_index":1,				
}
pong_or_kong_player_index：碰或者杠玩家索引	
cur_player_index：当前出牌玩家索引
